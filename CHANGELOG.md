## [1.0.0] - 2026-07-24

### Added

- Initial public release.
- Live visualization of active trading strategies directly on the chart.
- Support for Manual, ATM, and NinjaScript strategies.
- Strategy identity and snapshot reconciliation engine.
- Personal and Aggregated strategy display modes.
- Average price markers with optional unrealized P&L display.
- Entry, Stop, and Target order visualization.
- Automatic synchronization with ChartTrader account and instrument selection.
- Configurable NinjaScript stop/target signal name recognition.
- Debug snapshot logging for reconciliation diagnostics.
- SharpDX rendering with cached Direct2D resources for improved performance.
- Comprehensive project documentation and installation guide.
- PolyForm Noncommercial License 1.0.0.
